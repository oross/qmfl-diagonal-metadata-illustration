# Supplementary material

Supplementary material for:

**An Effect-Algebraic Framework for Mediative Truth--Falsity Semantics in Quantum Evidence Models**

This repository contains the minimal files needed to reproduce the numerical values reported in the diagonal metadata illustration of the manuscript. The material is semantic and reproducibility-oriented. It does not train, validate, or benchmark an autonomous-driving perception model, and it does not claim empirical superiority, safety performance, or quantum computational advantage.

## Contents

1. `zod_selected_preprocessed_rows.csv`

   Minimal preprocessed representation of the three ZOD-derived metadata rows used in the manuscript. The file contains only the fields needed to reproduce the illustrative quantities reported in the table: split, frame identifier, metadata descriptors, contextual coding components, contextual-complexity score inputs, distance when available, and the truth--falsity quantities used in the diagonal embedding.

2. `reproduce_zod_mfl.py`

   Minimal Python script that recomputes the contextual-complexity index `c`, the intermediate distance/proximity quantities when applicable, the truth--falsity pair `(mu_p, nu_p)`, the diagnostic quantities `(pi_p, zeta_p)`, and the mediative value `M_q(p,rho)` reported in the manuscript.

   The script uses only the Python standard library.

3. `zod_mfl_recomputed_results.csv`

   Output file generated by the script. It contains the recomputed illustrative values used to check the numerical entries reported in the manuscript.

4. `reproduce_zod_mfl_notebook_v5.ipynb`

   Annotated notebook version of the same calculation. It is provided only for readability and step-by-step inspection of the semantic computation.

5. `LICENSE.txt`

   MIT License for the supplementary code.

## How to reproduce

Place the CSV file and the Python script in the same folder and run:

```bash
python reproduce_zod_mfl.py
```

The script writes:

```bash
zod_mfl_recomputed_results.csv
```

and prints the rounded values corresponding to the illustrative table in the manuscript.

## Quantities reproduced by the script

The contextual-complexity index is computed as

```text
c = 0.35*c_weather + 0.25*c_light + 0.20*c_road + 0.20*c_density.
```

For rows with a positive metadata risk indicator, distance is converted into

```text
D = clip(d/10, 0, 1),
r = 1 - D.
```

The illustrative truth--falsity pair is then computed as

```text
mu_p = 0.55 + 0.35*r + 0.10*c,
nu_p = 0.05 + 0.20*D + 0.25*c.
```

For rows without a positive metadata risk indicator, the illustrative truth--falsity pair is computed as

```text
mu_p = 0.10 + 0.30*c,
nu_p = 0.60 + 0.25*(1 - c).
```

The diagnostic quantities are

```text
pi_p   = max(0, 1 - mu_p - nu_p),
zeta_p = max(0, mu_p + nu_p - 1).
```

The corresponding mediative value is

```text
M_q(p,rho) = (1 - pi_p - zeta_p/2)*mu_p
             + (pi_p + zeta_p/2)*(1 - nu_p).
```

In the manuscript, this value is interpreted through the diagonal embedding

```text
M_q(p,rho) = M(mu_p,nu_p).
```

The computation is purely illustrative. The metadata-derived values first define classical truth--falsity pairs, and the diagonal embedding realizes those pairs as Born expectations of quantum effects. Non-commuting effects are not required for this diagonal instantiation.

## Data note

The selected rows are derived from preprocessed ZOD metadata and are used only to reproduce the illustrative diagonal metadata calculations in the manuscript. They do not contain raw ZOD sensor data, images, LiDAR, radar, video, trained perception models, or personally identifying information.

The original Zenseact Open Dataset remains governed by its own license terms. Users should consult and cite the original ZOD source when using or referencing ZOD-derived material.

## Scope note

This supplementary material does not provide an autonomous-driving system, a perception benchmark, a detector, a safety policy, or an empirical validation protocol. It only reproduces the semantic quantities used in the diagonal metadata illustration of the manuscript.
